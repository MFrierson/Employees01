
Git commmands:


change user name:  git config --global user.nam "Mac"

change user email:  git config --global user.email "blacklantern78@yaho.com"

show version:  git --version

Show current directory:  pwd

Change Directory:  cd /c/Users/Bruce/Desktop/Projects
(you can drag the target folderin too)


Create Directory: mkdir "new folder"


initialize Local Git Repository(should only have to do this once): $ git init

add file to index(staging area before commit): $ git add <file>
 (example: $ git add index.html)

add all of a certain type of file to index: $ git add *.html

add all files in a directory to index: $ git add .


remove file(s) from index(staging area before commit): $ git rm --cached <file>
 (example: $ git rm --cached indedx.html)


show what is in the staging area: $ git status


Commit the changes in the index: $ git commit
- press I to enter "Insert Mode"
- press Esc - to exit "Insert Mode"
- press :wq + enter = Final commit

Commit the changes in One line: $ git commit -m 'some comment'
  (where some comment = comment for commit(keep the ''))


Create a new branch: $ git branch <name of branch>

Change to the new branch: $ git checkout <name of branch>

Merge branch into master(from master): get merge <name of branch>


List remote respoistories: $ git remote

First time Push(you want to pull for when creating the directory first):

Run: git remote add <name> <"url for repository from github">
Then Run: git push -u <name>master
 (Log into Git, and create the repository with the same name as the repository/MainFolder)
 (This will get you the URL for the repository)

Connect to a specific remote repository: $ git remote add Employees01 <"url for repository from github">
 (where 'origin' is the name you are giving the remote on your local computer)


Push local repository to Git Hub: $ git push
  (example: $ git push -u Employees01 master)
-(this will prompt a login screen from GitHub)
-(after the first push, you just have to use "$ git push")




Pull the changes from the Git repository to the local one: $ git pull
	
Clone Repository to a New Directory: $ git clone
  (example: $ git clone <"url for repository from github">)

Create a new .gitignore file: $ touch .gitignore
(you would not be able to create the file in normal Windows OS)
(edit the file in a text editor)
